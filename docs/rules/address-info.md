# Address Info Rules (SYNTHETIC — demo only)

Mirrors `server/rules/address.js`.

- Must be a Canadian residential address — PO boxes are not accepted.
- Supported provinces: ON, BC, AB, QC, MB, NS, NB, SK, PE, NL.
- Postal code must match the standard Canadian format (e.g. M5V 1M1).
- The province captured here drives which provincial disclosures appear later, in Tax questions and Terms & conditions.
